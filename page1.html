Dev Log #1: Project Kickoff / Announcement

April 17, 2025

I have alwyas loved playing games and over the years I have spent time developing games but dont really have much to show for it. I was hit with some inspiration to finally finish a project on my own as a solo developer. So fuck it I started this project. The direction of the game isnt super clear at the moment, but it will be a multiplayer game with horror elements and a focus on exploration/extraction. I have been a bit inspired by Indiana Jones and more recently from reading "King Solomon's Mines" by H. Rider Haggard and "The Lost City of Z" by David Grann.

Key achievements so far:

  • ✓ Playable Character ~ With working health and stamina
  • ✓ Basic item mechanics ~ Working Inventory system and backpack plus a working flashlight
  • ✓ Expanded game mechanics ~ Item pick up/drop, loot objects to collect, interactable objects
  • ✓ Basic Traps ~ Triggerable pressure plate that can activate traps (a boulder so far)
  • ✓ Death Mechanic ~ Player can die from taking damage. Triggers ragdoll, spotlight, and camera change.
