Senior Project

System consisting of three main components. The first component is the robot that is able to gather lidar data from a building. This second component, the server, is responsible for using the lidar data to create a map and also for communication between component 1 and 3. The last component is the Unreal Engine client which allows users to see a virtual world made from information gathered from the robot.
